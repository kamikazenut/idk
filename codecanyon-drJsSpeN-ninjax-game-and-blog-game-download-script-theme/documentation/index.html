Installation


Step 1: Instal local server XAMPP
Step 2 : Extract Ninjax.zip File to Desktop.
Step 3 : Run your server and go to http://localhost/phpmyadmin.

Step 4 : Create database name Ninjax.

Step 5 : import database From Files Database. (Ninjax.sql)

Step 6 : run your Project. : php artisan serve

Enjoy your project

Note : if you want to change database name go to .ENV file and change name
